# 个人信息助手 (Personal Information Assistant)

一个功能强大的浏览器油猴脚本，提供侧边栏形式的个人信息管理助手，支持分类管理、搜索、拖拽排序等高级功能。

## 📋 项目概述

个人信息助手是一个专为浏览器设计的用户脚本，通过油猴插件运行。它提供了一个美观、实用的侧边栏界面，帮助用户高效管理个人简历信息、学习笔记、工作记录等各类信息。

### 核心特性
- 🎯 **智能分类管理** - 支持多级分类，灵活组织信息
- 🔍 **实时搜索** - 快速定位所需信息
- 🖱️ **拖拽排序** - 直观的信息项排序功能
- 📱 **响应式设计** - 适配不同屏幕尺寸
- 💾 **本地存储** - 数据安全保存在本地
- 🎨 **美观界面** - 现代化的UI设计

## 🚀 快速开始

### 环境要求
- 支持油猴插件的浏览器（Chrome、Firefox、Edge等）
- 已安装油猴插件（Tampermonkey或类似插件）

### 安装步骤

1. **安装油猴插件**
   - 在浏览器扩展商店中搜索并安装 Tampermonkey

2. **安装用户脚本**
   - 打开油猴插件管理界面
   - 选择"创建新脚本"
   - 将 `resume_assistant.js` 文件内容复制到编辑器中
   - 保存脚本

3. **启用脚本**
   - 确保脚本已启用
   - 刷新任意网页即可看到侧边栏

## 📖 使用指南

### 基本操作

#### 1. 打开/关闭侧边栏
- **展开侧边栏**: 点击最小化状态下的侧边栏图标
- **最小化侧边栏**: 点击标题栏的关闭按钮（×）
- **切换位置**: 点击切换按钮（◀/▶）在左右侧切换

#### 2. 添加信息
1. 点击底部"+ 添加信息"按钮
2. 填写标题、内容、日期和分类
3. 点击"保存"完成添加

#### 3. 分类管理
- **添加分类**: 点击分类区域的"+"按钮
- **切换分类**: 点击分类按钮查看对应信息
- **重命名分类**: 右键点击分类选择"重命名"
- **删除分类**: 右键点击分类选择"删除"

#### 4. 搜索功能
- 在底部搜索框中输入关键词
- 系统将实时筛选匹配的信息项

### 高级功能

#### 拖拽排序
- **信息项排序**: 直接拖拽信息项调整顺序
- **分类排序**: 拖拽分类按钮调整分类顺序

#### 右键菜单
- **编辑信息**: 右键点击信息项选择"编辑"
- **删除信息**: 右键点击信息项选择"删除"
- **查看详情**: 点击信息项查看完整内容

## 🔧 API 接口说明

### 数据存储结构

```javascript
// 应用数据结构
{
    isFixed: boolean,           // 是否固定位置
    sidebarPosition: 'left'|'right', // 侧边栏位置
    collapsedPosition: number,   // 最小化位置
    categories: string[],        // 分类列表
    items: [                     // 信息项列表
        {
            title: string,       // 标题
            content: string,     // 内容
            category: string,    // 分类
            startDate: string,   // 开始日期
            endDate: string,     // 结束日期
            id: string,          // 唯一标识
            order: number        // 排序序号
        }
    ]
}
```

### 核心函数

#### `initializeStorage()`
初始化本地存储结构，确保数据格式正确。

#### `loadData()`
从本地存储加载数据，支持向后兼容旧格式。

#### `saveData()`
保存当前数据到本地存储。

#### `generateId()`
生成唯一标识符，用于信息项和分类。

#### `renderCategories()`
渲染分类列表到界面。

#### `renderItems(category)`
渲染指定分类的信息项到界面。

#### `addItem(itemData)`
添加新的信息项。

#### `editItem(itemId, newData)`
编辑指定ID的信息项。

#### `deleteItem(itemId)`
删除指定ID的信息项。

#### `addCategory(categoryName)`
添加新的分类。

#### `deleteCategory(categoryName)`
删除指定分类及其所有信息项。

## ❓ 常见问题解答

### Q: 侧边栏不显示怎么办？
A: 检查油猴插件是否启用，脚本是否在目标网站上运行。

### Q: 数据丢失了如何恢复？
A: 数据自动保存在浏览器本地，重新安装脚本后会自动加载。

### Q: 如何备份我的数据？
A: 目前数据自动保存在浏览器中，如需手动备份可导出存储数据。

### Q: 支持移动端吗？
A: 脚本主要针对桌面浏览器设计，移动端体验可能有限。

### Q: 可以自定义分类颜色吗？
A: 当前版本使用统一配色，未来版本可能支持自定义。
